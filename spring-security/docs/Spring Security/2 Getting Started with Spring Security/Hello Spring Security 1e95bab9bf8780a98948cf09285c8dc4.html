  • Use the chapter02.00-calendar project as the base.

✅ 2. Add Spring Security Dependencies (Gradle)



implementation 'org.springframework.security:spring-security-config'
implementation 'org.springframework.security:spring-security-web'

🔸 Purpose:

  • spring-security-config: Core APIs and DSLs for configuring Spring Security.
  • spring-security-web: Servlet filter and web integration layer.

✅ 3. Implement Java-Based Security Configuration

🗂️ File: SecurityConfig.java

  • Uses @Configuration + @EnableWebSecurity.
  • Defines a user using InMemoryUserDetailsManager.
  • Sets a SecurityFilterChain bean to define rules.

🔸 Highlights:

  • All pages require ROLE_USER.
  • Uses .formLogin() with defaults.
  • CSRF disabled temporarily.
  • H2 console headers disabled.

✅ This configuration provides:

  • Form login
  • Basic user authentication
  • Filter chain setup
  • Minimal secure web app

✅ 5. The Security Filter Chain Explained

  • Spring Security uses servlet filters to control access.
  • SecurityFilterChain creates a filter bean called springSecurityFilterChain.
  • All HTTP requests pass through this filter for:
    • Authentication
    • Authorization
    • CSRF
    • Session management

🧠 Analogy: Like airport security—every request must pass through it.

✅ 6. WebAppInitializer: Bootstrapping the Security Filter

🗂️ File: SecurityWebAppInitializer.java



@Order(1)
public class SecurityWebAppInitializer extends 
AbstractSecurityWebApplicationInitializer {}

🔍 Why it matters:

  • Registers springSecurityFilterChain.
  • Ensures it’s the first filter in the chain.
  • Loaded using ContextLoaderListener.

✅ 7. Servlet Context & Filter Setup – Key Concepts

  • DispatcherServlet: Loads Spring MVC-specific beans (controllers, view resolvers).
  • ContextLoaderListener: Loads Spring Security beans (filters, authentication manager).

📌 Important:

Spring MVC beans can access Security beans, but not vice versa.

✅ 8. DelegatingFilterProxy

  • A Servlet Filter that delegates work to a Spring-managed bean.
  • Key to integrating Spring Security into any web container.

💡 Spring Boot users don’t need to worry—it's auto-configured.

But in pure Spring MVC apps, this setup is essential.

✅ 9. FilterChainProxy – The Security Enforcer

  • Receives requests from DelegatingFilterProxy.
  • Delegates to a chain of security filters:
    • UsernamePasswordAuthenticationFilter
    • ExceptionTranslationFilter
    • SecurityContextPersistenceFilter
    • ...and more.

🔁 Applies filters conditionally depending on the request.
