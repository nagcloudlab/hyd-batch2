V1 - No Security Demo

Problem: ALL endpoints are accessible to everyone! No authentication or authorization. In V2, we will add Spring Security with session-based login.

Endpoints in this app:

  • PUBLIC / - Home page
  • PUBLIC /about - About page
  • PUBLIC /api/public/greeting - Public REST API
  • PRIVATE /dashboard - Dashboard (sensitive data!)
  • PRIVATE /admin - Admin panel (sensitive data!)
  • PRIVATE /profile - User profile (sensitive data!)
  • PRIVATE /api/private/users - Private REST API

Try clicking on the private endpoints - you'll see they are wide open!
